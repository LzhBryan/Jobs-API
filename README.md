
# Jobs API

- An express app that performs CRUD operations on jobs along with JWT authentication and tested routes.

- [Live Preview](https://jobsapi.adaptable.app)
## Tech Stack

- Node
- Express
- TypeScript
- MongoDB
- Jest
- Supertest
- Swagger
- Express Validator
